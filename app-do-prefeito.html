APP DO PREFEITO

Clique aqui!  

É um software de gerencimento municipal, é contemplado por um site com folha de rosto e uma api de busca de dados em sites governamentais, presente em mais de 20 municípios.

Oferece:


Responsividade

Notícias regionais

Dados Atualizados

Salvar dados

Consultas Financeiras

Relatórios

Buscar Dados

Ambiente Privado


Utilizou se em seu desenvolvimento a linguagem de programação JAVA com o Framework SPRING BOOT. Para a arquitetura usou-se REST, a exibição na view é demonstrada com a bilbioteca THYMELEAF, além da biblioteca HTMLUNIT para busca de dados e a api JAVAMAIL para o envio de emails. Para o gerenciamento do banco de dados usou-se o MYSQL. Para servidor de aplicações utilizou se o TOMCAT. Atualmente a aplicação está rodando no servidor de hospedagem UBUNTU SERVER da AWS.
